Store Filters

Overview

Store filters consist of one or more conditions. The conditions are described in detail below.

Store List

The Store List condition allows the inclusion or exclusion of one or more specific stores.

Field

The Field condition allows comparison for store fields.

Status

The Status condition allows the inclusion of exclusion of one more store statuses.

Characteristics

The Characteristic condition allows comparision against store characterisitcs.

Attribute Set

The Attribute Set conditions allows for the inclusion or exclusion of store characteristics within the attribute set.

Variable to Constant

The Variable to Constant condition allows comparision between a variable and a literal value.

Variable to Variable

The Variable to Variable condition allows comparision between a variable and another variable.

Variable Percentage

The Variable Percentage condition allows comparision of percentage values between two variables.

Sort

The Sort condition allows the result set to be sorted.
